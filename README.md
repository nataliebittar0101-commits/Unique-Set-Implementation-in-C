# Unique Set Implementation in C

This project is a C program that reads integer numbers from the user, stores only unique values, and prints the resulting set.

The goal of the project is to practice core C programming concepts such as pointers, arrays, dynamic memory allocation, input handling, and duplicate removal.

## Features

- Reads integer input from the user
- Stores only unique values
- Removes duplicate numbers
- Uses dynamic memory allocation with `malloc` / `realloc`
- Prints the set in the order of insertion

## Technologies

- C
- Pointers
- Dynamic Memory Allocation
- Arrays
- Standard Input / Output
